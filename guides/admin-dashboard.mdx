The Stampchain admin dashboard is your central hub for managing loyalty campaigns, monitoring performance, and accessing all platform features. This guide provides a comprehensive overview of the dashboard interface and capabilities.

## Dashboard Overview

The admin dashboard is located at `/admin` and serves as the main control center for your Stampchain account. It provides real-time statistics, campaign management tools, and access to analytics.

**Location in code:** `src/app/(protected-route)/admin/page.tsx:20`

## Dashboard Layout

The dashboard is organized into three main sections:

1. **Stats Overview**: Key metrics displayed at the top
2. **Tabbed Navigation**: Three tabs for Campaigns, Analytics, and Settings
3. **Content Area**: Tab-specific content and management tools

## Stats Overview

At the top of the dashboard, you'll see two prominent statistics cards that provide a quick snapshot of your account performance.

**Location in code:** `src/app/(protected-route)/admin/components/stats.tsx:139`

### Total Campaigns

The first card displays:
- **Total number of campaigns** created in the current month
- **Percentage change** from the previous month
- **Trend indicator** (up or down arrow)
- Icon: Target symbol

Example: "12 Total Campaigns" with "15% from this month" and an upward trend arrow.

**Location in code:** `src/app/(protected-route)/admin/components/stats.tsx:98`

### Tokens Distributed

The second card shows:
- **Total tokens distributed** through all campaigns
- **Percentage change** from last month
- **Trend indicator** showing growth or decline
- Icon: Coins symbol

Example: "2,340 Tokens Distributed" with "22% from last month" and an upward trend arrow.

**Location in code:** `src/app/(protected-route)/admin/components/stats.tsx:119`

<Info>
  These statistics are calculated in real-time from your database and update automatically as campaigns are created and tokens are claimed.
</Info>

<Steps>
  <Step title="Click 'New Campaign' Button">

### Generating QR Codes

<Steps>

## Performance Optimization

The dashboard uses several techniques to ensure fast loading:

- **Suspense Boundaries**: Components load independently
- **Skeleton Loaders**: Show loading states instead of blank screens
- **Data Caching**: Frequently accessed data is cached
- **Lazy Loading**: Charts and graphs load on demand

## Troubleshooting

### Dashboard Won't Load

1. Check your authentication status
2. Verify internet connection
3. Clear browser cache
4. Try incognito/private mode
5. Check browser console for errors

### Stats Not Updating

1. Refresh the page
2. Check if campaigns have recent activity
3. Verify database connection
4. Wait a few minutes for cache to clear

### Campaign Details Missing

1. Ensure campaign exists in database
2. Check campaign ID in URL
3. Verify you have permission to view
4. Try accessing from campaign list
